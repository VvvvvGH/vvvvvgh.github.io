    记博客更新

    距离上次博客更新已经过了许久，周末心血来潮想重整一下，做一个扫除，也想把这些年写的东西和内容做个更新。除了博文，还想把这些年走过的地方和照片更新一下，放到网站里，类似一个相册，做成备份，当然了，不涉及私人内容，只放一些随机风景照和记录。这些年应该少有人再做这种传统的博客了吧，不过呢，我就喜欢这种DIY 的。

    那么，这次整了哪些东西呢？

    • Blog 程序： https://vuepress.vuejs.org/
    • 主题：vuepress-theme-hope
    • 图床：
      • 腾讯云 COS存储
      • PicList 客户端
      • 默认 webp 压缩
    • 部署： 使用 Github action 从私有仓库部署到 Github page
    • 编辑器 Obsiden：
      • 变更自动同步代码仓库，自动触发更新
